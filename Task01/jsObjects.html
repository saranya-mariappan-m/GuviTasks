Objects in Javascript

What is Objects?

Objects are like arrays in Javascript which can hold different primitive types of data. An object is a collection of related data and/or functionality. These usually consist of several variables and functions

It is a common practice to declare objects with the const keyword.

Creating Objects in Javascript

  • By object literal
  • By creating instance of Object directly (using new keyword)
  • By using an object constructor (using new keyword)

1) JavaScript Object by object literal

object={property1:value1,property2:value2.....propertyN:valueN}

Example: const person = {}; ie; const emp={id:102,name:"Shyam Kumar",salary:40000}

2) By creating instance of Object

var objectname=new Object();

Example: var emp=new Object(); emp.id=101; emp.name="Ravi Malik"; emp.salary=50000;

3) By using an Object constructor

create function with arguments. Each argument value can be assigned in the current object by using this keyword.

The this keyword refers to the current object.

Example: function emp(id,name,salary){ this.id=id; this.name=name; this.salary=salary; } e=new emp(103,"Vimal Jaiswal",30000);

Dot Notation:

Example: person.age

Built in Objects

Example: const currentDate = new Date();

Internal Representation of Objects

If we create a Object, const obj = {'a':1;'b':2}; When we compile the code either on browser or on Node, At first memory is not allocated for obj constant, If we asssigned it as 'var' then the memory would be allocated on the Global Frame/Scope. Once we execute the initialized const obj, then the memory is allocated for the obj with referencing the scope object for a = 1 and b = 2. Global scope will be persisting. Functional scope is volatile.
