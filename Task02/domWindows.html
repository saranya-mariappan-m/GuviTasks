Difference between DOM and Windows in Javascript

What is Windows?

The window object is the topmost object of the DOM hierarchy. It represents a browser window or frame that displays the contents of the webpage. Whenever a window appears on the screen to display the contents of the document, the window object is created.

Window Properties:

window.property_name;

Example: window.innerWidth;

Methods of Windows:

window.method_name;

Example: window.setTimeout();

What is Document/DOM(Document object model)?

A document is an object inside the window object and we use a document object for manipulation inside the document.

The first thing that gets loaded into the browser is the window and the properties related to that window are stored in the window object. Properties related to window objects are length, innerWidth, innerHeight, caches, etc. After the window gets loaded then thereâ€™s a document (HTML, PHP, or another document) loaded inside that window, and the properties related to that document is stored in the document object

Document Properties:

document.propertyname;

Example: document.domain;

Methods of Document:

document.method_name;

Example: document.getElementById();
